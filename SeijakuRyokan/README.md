# SeijakuRyokan

![ゲームスクリーンショット](https://user-images.githubusercontent.com/13523679/163682893-314b6c1a-82f3-444c-8a9a-4fa274a9a1c9.png)

**製作者:** AptmaSTUDIO

## 概要

『SeijakuRyokan』は、静かな旅館を舞台にしたパズル＆マネジメントゲームです。プレイヤーは旅館の新人スタッフとなり、次々と訪れるお客様を目的の部屋までスムーズにご案内することが目的です。衝立（ついたて）や案内板をうまく配置して、お客様同士が鉢合わせしないように、最適な動線を作り出しましょう。

## ゲームの目的

お客様を時間内に目的の部屋へご案内し、満足度（スコア）を高めることが目標です。お客様にはそれぞれ機嫌やタイムリミットがあり、他のお客様とすれ違うと不機嫌になってしまいます。効率的な経路を設計し、ハイスコアを目指しましょう。

## 主な機能

*   **建築システム:**
    *   **廊下:** お客様が歩くための基本的な道を作ります。
    *   **衝立:** お客様の視線を遮り、鉢合わせを防ぎます。ただし、設置にはコストがかかります。
    *   **案内板:** お客様を特定の方向に誘導します。向きを調整することが重要です。
*   **お客様のAI:**
    *   お客様はそれぞれ目的地を持ち、最短経路を探索して移動します。
    *   他のお客様の視界に入ると、満足度が低下します。
*   **スケジュール管理:**
    *   お客様はスケジュールに沿って旅館を訪れます。画面左下のキューで次のお客様を確認し、準備を整えましょう。
*   **スコアと期限:**
    *   お客様を無事に部屋へ案内できるとスコアが加算されます。
    *   お客様ごとに設定された期限を過ぎると、ゲームオーバーになる可能性があります。

## 操作方法

このゲームはゲームパッドでのプレイに最適化されています。

| アクション           | ゲームパッド       |
| -------------------- | ------------------ |
| カーソル移動         | 左スティック       |
| 建築モード切替       | LB / RB            |
| 建築・決定           | Aボタン            |
| 撤去                 | Aボタン（対象の上で） |
| 案内板の向き変更     | 方向キー（←/→）  |
| 待機中のお客様を入場 | Xボタン            |
| ヘルプ表示           | SELECTボタン       |

## 実行環境

*   **OS:** Windows 10/11
*   **グラフィックス:** DirectX 11 対応グラフィックスカード

## ビルド方法

Microsoft Visual Studio 2022 と C++17 対応のコンパイラが必要です。
`SeijakuRyokan.sln` を開き、`SeijakuRyokan` プロジェクトをビルドしてください。
