======================================
        Base プログラム 使用方法
======================================

本ソフトウェアは、任天堂 Switch 開発環境で動作する描画基盤です。
課題制作・研究用に、下記の手順に従って利用してください。

---

【1. 必要環境】
--------------------------------------
- 任天堂 Switch 開発キット（ライセンス必須）
- 任天堂 Switch 開発SDKがインストールされているPC
- Visual Studio 2017以降（v143 推奨）
- vcpkg（事前にインストール、integrate install 済み）

---

【2. ビルド方法】
--------------------------------------
1. Visual Studio でソリューションを開く
2. 構成を選択：
   - PC_Debug … Windows上でのデバッグ用
	- x64に設定してください
   - Switch_Debug … Switch上での動作確認用
	- NX64に設定してください
3. [ビルド] → [ソリューションのビルド] を実行

※ ビルドエラーが出る場合は、プロジェクトの「ターゲットプラットフォーム」を再ターゲットし、v143 に変更してください。

---

【3. プログラム概要】
--------------------------------------
- エントリポイント：`pc_main.cpp`/`switch_main.cpp`
- 描画初期化処理：`InitSystem()`
- スプライト描画処理：`DrawSpriteTriangle()`, `DrawSpriteQuad()`
- 終了処理：`UninitSystem()`

任天堂 SDK の API に基づいて、テクスチャ読み込み、描画、バッファスワップを行っています。

---

【4. プログラムの改変について】
--------------------------------------
- 本ソフトウェアは課題・研究目的に限り、自由に改変・カスタマイズ可能です。
- 改変後の成果物にも、元ソフトウェアの利用と作者名のクレジットを記載してください。
- 新しくプログラム（ソースやヘッダ）を追加する際は、必ず「PC_Debug 構成」または「Switch_Debug 構成」のどちらでビルドするかを確認し、不要な構成では 
**ビルドから除外する設定** をしてください。
  （ソリューションエクスプローラーでファイルを右クリック→「プロパティ」→「構成ごとのビルドに含める」をOFFにできます）
